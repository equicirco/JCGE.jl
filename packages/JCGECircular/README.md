# JCGECircular

Circular-economy extension blocks for JCGE.

## Responsibilities
- Use/service composite demand blocks (e.g., new/refurb/repair/rental)
- Refurbishment, repair, rental provision blocks
- Returns/scrap accounting hooks
- Optional circular policy instruments/constraints

## Dependencies
- Depends on JCGECore, JCGECalibrate, JCGEKernel (and possibly JCGEBlocks)
